## Para rodar Localmente

Clone o projeto

```bash
  git clone git@github.com:maricegalla/portfolio.git
```

Abra o diretório do projeto

```bash
  cd portfolio
```

Instale as dependências do projeto

```bash
  npm install
```

Inicie a Aplicação

```bash
  npm start
```

## Stack utilizada

**Front-End:** ReactJS & Styled Components.
